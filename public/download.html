Dependencies

In order to use this program you need to have python 3.x installed and accessible on your laptop (in your path environment variable).
If you haven’t a proper installation of python and/or the following libraries the program will not compute anything, but the interface will still work.

Library needed :

  • Numpy
  • Sympy

Windows

The easiest way is to install the scientific pre-pack Anaconda program.
Or via pip : 
pip install numpy sympy

Mac/Linux

Via pip : 
pip install numpy sympy
