About this program:

This program compute raman selection rules depending on :

  1. The symmetry of the crystal
  2. The orientation of incident and scattered polarizations
  3. The Pointing vector
  4. Polarization of your excitation/collection path
Calculations are done in back scattering geometry (same Pointing vector for incident and scattered light but with opposite direction)


The reference axis is needed to compute polarisation depending on the Pointing vector. (see "How to use" section for details)
It must be perpendicular to the Pointing vector


You can find more information about how to use this software in the "How to Use" rubric.
